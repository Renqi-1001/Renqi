# Appendix for "Rate-Splitting Grant-Free Transmission for Uplink Coexistence in LEO Satellite Networks"

This repository contains the supplementary materials and detailed mathematical proofs for the conference paper **"Rate-Splitting Grant-Free Transmission for Uplink Coexistence in LEO Satellite Networks"**.

## Contents

* **`Appendix.pdf`**: This document provides the complete mathematical derivations, including detailed system modeling and signal processing proofs, which were omitted from the main conference paper due to space constraints.

## Note to Reviewers and Readers

You can view or download the PDF directly from this repository to access the full proof details. If you have any questions regarding the derivations, please feel free to contact the authors.
